EMF Bot

About the App

EMF Bot is a desktop application developed in Python for Windows, designed to automate the sending of emails, either individually or in a loop with a time interval defined by the user. It's a useful tool for testing purposes, controlled bulk sending, or automated notifications in personal or development environments.

The application uses Python's standard "smtplib" library, which enables connections to mail servers via the SMTP, also known as "Simple Mail Transfer Protocol", ensuring the structured delivery of messages from configured accounts. For its graphical interface, EMF Bot relies on "tkinter", Python's native library for building simple and functional user interfaces, allowing users to input recipient addresses, subject lines, message content, sending frequency, and other basic settings.
