Standard Calculator

About the Project

This website has been created as part of a practical exercise to develop and strengthen basic skills in HTML, CSS, and JavaScript. As its name suggests, it is a simple calculator, designed to provide the basic functionalities of a conventional calculator, such as addition, subtraction, multiplication, and division.
Unlike a basic calculator with no design, this version features a more visually appealing interface, achieved through the use of custom CSS, which enhances the user experience with a modern and visually pleasing style.
As for its internal workings, all operations and results are handled through simple JavaScript functions, allowing for dynamic and efficient interaction without the need to reload the page.
