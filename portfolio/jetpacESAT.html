Jetpac Spectrum

About the Game

This version of the classic game Jetpac, originally released for the ZX Spectrum, has been developed as a practical project aimed at applying skills acquired in C and C++ programming. The game was created using ESAT's proprietary graphics library, which enabled the development of a visual and playable experience faithful to the original, but with an academic focus.

The game offers two gameplay modes: single-player and two-player. In both cases, the gameplay remains true to the core essence of the original Jetpac. The main objective is to rebuild a spaceship by assembling its parts, and then refuel it. Once this task is completed, the player can progress to the next level.

During gameplay, players must collect pick-ups that appear on screen while avoiding or eliminating enemies trying to stop their progress. The action is constant and demands quick reflexes and strategic thinking in order to survive and advance through the game.
