God Arc Nexus

About the Project

God Arc Nexus is an Android application developed as an academic project with the goal of demonstrating the skills acquired in the use of Android Studio, the Kotlin programming language, and the handling of text strings through XML files. The app functions as an interactive wiki focused on the video game God Eater 3 by Bandai Namco.

The application is structured into several sections. It begins with a Login screen, which, while it does not store real user data, simulates the behavior of an authentication interface. After logging in, the user is taken to the Home screen, which provides a general overview of the game's story.

From the main menu, users can navigate to three primary sections: Characters, Weapons and Aragamis. Each of these sections features an interactive list, allowing users to select individual items to access a detailed view with more information about the selected character, weapon, or creature.

As a technical enhancement, God Arc Nexus includes basic Firebase integration, used to collect information about potential errors or crashes during runtime. This helps ensure more efficient performance monitoring and greater application stability.
