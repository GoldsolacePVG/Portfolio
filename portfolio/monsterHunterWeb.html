Monster Hunter Web

About the Project

This website was developed as the final project for the Web Programming course in the Videogame Programming degree at ESAT. Its purpose is to demonstrate mastery of the fundamental knowledge acquired during the course in the use of HTML, CSS, and JavaScript, applied to the design and development of websites.

The project consists of a themed website centered around the Monster Hunter video game series. Throughout the site, users can explore different sections dedicated to various aspects of the game universe. These include a general introduction, a timeline that traces the evolution of the series, a page focused on the most iconic monsters, another showcasing the different weapon types available in the games, and finally, a section about the lore and narrative background of the Monster Hunter universe.

In addition to these main sections, the website features a Contact page, which allows users to send inquiries or suggestions, and a Privacy Policy, created as part of best practices in web development. As a final touch, the site includes a section dedicated to the developer, presenting a first version of the author's CV.

This project not only reflects the technical skills acquired during the course, but also the ability to structure content, create a coherent user experience, and present information in a visually appealing way.
